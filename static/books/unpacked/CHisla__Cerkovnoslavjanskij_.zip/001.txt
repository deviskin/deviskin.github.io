И҆ гл҃а гдⷭ҇ь къ мѡѷсе́ю въ пꙋсты́ни сїна́йстѣй, въ ски́нїи свидѣ́нїѧ, въ пе́рвый де́нь мцⷭ҇а втора́гѡ, втора́гѡ лѣ́та и҆зше́дшымъ и҆̀мъ ѿ землѝ є҆гѵ́петскїѧ, гл҃ѧ:
возми́те сочте́нїе всегѡ̀ со́нма сынѡ́въ і҆и҃левыхъ по сродствѡ́мъ и҆́хъ, по домѡ́мъ ѻ҆те́чества и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ:
всѧ́къ мꙋ́жескъ по́лъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою і҆и҃левою, соглѧ́дайте и҆̀хъ съ си́лою и҆́хъ, ты̀ и҆ а҆арѡ́нъ созира́йте и҆̀хъ:
и҆ съ ва́ми да бꙋ́дꙋтъ съ си́лою свое́ю кі́йждо, кі́йждо по пле́мени коегѡ́ждо ѿ кнѧзе́й, по домѡ́мъ ѻ҆те́чествъ да бꙋ́дꙋтъ.
И҆ сїѧ̑ и҆мена̀ мꙋже́й, и҆̀же ста́нꙋтъ съ ва́ми: ѿ рꙋви́ма є҆лїсꙋ́ръ сы́нъ седїꙋ́ровъ,
ѿ сѷмеѡ́на саламїи́лъ сы́нъ сꙋрїсада́евъ,
ѿ і҆ꙋ́ды наассѡ́нъ сы́нъ а҆мїнада́вль,
ѿ і҆ссаха́ра наѳанаи́лъ сы́нъ сѡга́ровъ,
ѿ завꙋлѡ́на є҆лїа́въ сы́нъ хелѡ́нь,
ѿ сынѡ́въ і҆ѡ́сифовыхъ, и҆̀же ѿ є҆фре́ма, є҆лїса́мъ сы́нъ семїꙋ́довъ, ѿ манассі́и гамалїи́лъ сы́нъ фадассꙋ́ровъ,
ѿ венїамі́на а҆вїда́нъ сы́нъ гадеѡні́евъ,
ѿ да́на а҆хїе́зеръ сы́нъ а҆мїсада́евъ,
ѿ а҆си́ра фагаїи́лъ сы́нъ є҆хра́новъ,
ѿ га́да є҆лїса́фъ сы́нъ рагꙋи́ловъ,
ѿ нефѳалі́ма а҆хїре́й сы́нъ є҆на́нь.
Сі́и нарече́ннїи со́нма кнѧ̑зи ѿ племе́нъ, по ѻ҆те́чествамъ и҆́хъ, тысѧщенача́льницы і҆и҃лєвы сꙋ́ть.
И҆ поѧ̀ мѡѷсе́й и҆ а҆арѡ́нъ мꙋ́жы сїѧ̑ наречє́нныѧ и҆́менемъ.
И҆ ве́сь со́нмъ собра́ша въ пе́рвый де́нь мцⷭ҇а втора́гѡ лѣ́та, и҆ соглѧ́даша ѧ҆̀ по родѡ́мъ и҆́хъ, по ѻ҆те́чествамъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ мꙋ́жескъ по́лъ по глава́мъ и҆́хъ,
ꙗ҆́коже повелѣ̀ гдⷭ҇ь мѡѷсе́ю: и҆ соглѧ́дашасѧ сі́и въ пꙋсты́ни сїна́йстѣй.
И҆ бы́ша сы́нове рꙋви́ма, пе́рвенца і҆и҃лева, по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене рꙋви́млѧ четы́редесѧть ше́сть ты́сѧщъ и҆ пѧ́ть сѡ́тъ.
Сынѡ́въ сѷмеѡ́новыхъ по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене сѷмеѡ́нѧ пѧтьдесѧ́тъ де́вѧть ты́сѧщъ и҆ три́ста.
Сынѡ́въ і҆ꙋ́диныхъ по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене і҆ꙋ́дина се́дмьдесѧтъ четы́ри ты́сѧщы и҆ ше́сть сѡ́тъ.
Ѿ сынѡ́въ і҆ссаха́ровыхъ по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене і҆ссаха́рова пѧтьдесѧ́тъ и҆ четы́ри ты́сѧщы и҆ четы́ре ста̑.
Ѿ сынѡ́въ завꙋлѡ́нихъ по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене завꙋлѡ́нѧ пѧтьдесѧ́тъ се́дмь ты́сѧщъ и҆ четы́ре ста̑.
Ѿ сынѡ́въ і҆ѡ́сифовыхъ, сынѡ́въ є҆фре́млихъ по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене є҆фре́млѧ четы́редесѧть ты́сѧщъ и҆ пѧ́ть сѡ́тъ.
Ѿ сынѡ́въ манассі́иныхъ по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ и҆́хъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене манассі́ина три́десѧть и҆ двѣ̀ ты́сѧщы и҆ двѣ́сти.
Ѿ сынѡ́въ венїамі́новыхъ по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ и҆́хъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене венїамі́нѧ три́десѧть и҆ пѧ́ть ты́сѧщъ и҆ четы́ре ста̑.
Ѿ сынѡ́въ га́довыхъ, по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескїй по́лъ и҆́хъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене га́дова четы́редесѧть пѧ́ть ты́сѧщъ и҆ ше́сть сѡ́тъ и҆ пѧтьдесѧ́тъ.
Ѿ сынѡ́въ да́новыхъ по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ и҆́хъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене да́нова шестьдесѧ́тъ и҆ двѣ̀ ты́сѧщы и҆ се́дмь сѡ́тъ.
Ѿ сынѡ́въ а҆си́ровыхъ по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ и҆́хъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене а҆си́рова четы́редесѧть є҆ди́на ты́сѧща и҆ пѧ́ть сѡ́тъ.
Ѿ сынѡ́въ нефѳалі́млихъ по сродствѡ́мъ и҆́хъ, по со́нмѡмъ и҆́хъ, по домѡ́мъ ѻ҆те́чествъ и҆́хъ, по числꙋ̀ и҆ме́нъ и҆́хъ, по глава́мъ и҆́хъ, всѧ́къ мꙋ́жескъ по́лъ и҆́хъ ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й съ си́лою,
соглѧ́данїе и҆́хъ ѿ пле́мене нефѳалі́млѧ пѧтьдесѧ́тъ и҆ трѝ ты́сѧщы и҆ четы́ре ста̑.
Сїѐ сочисле́нїе, є҆́же соглѧ́даша мѡѷсе́й и҆ а҆арѡ́нъ и҆ кнѧ̑зи і҆и҃лєвы, двана́десѧть мꙋже́й, мꙋ́жъ є҆ди́нъ ѿ пле́мене є҆ди́нагѡ, по пле́мени домѡ́въ ѻ҆те́чества и҆́хъ бы́ша.
И҆ бы́сть всегѡ̀ соглѧ́данїѧ сынѡ́въ і҆и҃левыхъ съ си́лою и҆́хъ, ѿ два́десѧти лѣ́тъ и҆ вы́шше, всѧ́къ и҆сходѧ́й ѡ҆полча́тисѧ во і҆и҃ли,
ше́сть сѡ́тъ ты́сѧщъ и҆ трѝ ты́сѧщы и҆ пѧ́ть сѡ́тъ и҆ пѧтьдесѧ́тъ.
Леѵі́ти же ѿ пле́мене ѻ҆те́чества и҆́хъ не сочи́слишасѧ въ сынѣ́хъ і҆и҃левыхъ.
И҆ речѐ гдⷭ҇ь къ мѡѷсе́ю, гл҃ѧ:
ви́ждь, пле́мене леѵі́ина да не сочи́слиши, и҆ числа̀ и҆́хъ да не прїи́меши средѝ сынѡ́въ і҆и҃левыхъ:
и҆ ты̀ приста́ви леѵі́ты къ ски́нїи свидѣ́нїѧ и҆ ко всѣ̑мъ сосꙋ́дѡмъ є҆ѧ̀, и҆ ко всѣ̑мъ, є҆ли̑ка сꙋ́ть въ не́й: да но́сѧтъ сі́и ски́нїю и҆ всѧ̑ сосꙋ́ды є҆ѧ̀, и҆ ті́и да слꙋ́жатъ въ не́й, и҆ ѡ҆́крестъ ски́нїи да ѡ҆полча́ютсѧ:
и҆ внегда̀ воздвиза́ти ски́нїю, да снима́ютъ ю҆̀ леѵі́ти, и҆ внегда̀ поста́вити ски́нїю, да возста́вѧтъ: и҆ и҆ноплеме́нникъ пристꙋпа́ѧй да ᲂу҆́мретъ.
И҆ да ѡ҆полча́ютсѧ сы́нове і҆и҃лєвы, всѧ́къ въ свое́мъ чи́нѣ и҆ всѧ́къ по своемꙋ̀ старѣ́йшинствꙋ, съ си́лою свое́ю:
леѵі́ти же да ѡ҆полча́ютсѧ сопроти́въ, ѡ҆́крестъ ски́нїи свидѣ́нїѧ, и҆ не бꙋ́детъ согрѣше́нїѧ въ сынѣ́хъ і҆и҃левыхъ, и҆ да стрегꙋ́тъ леѵі́ти са́ми стра́жꙋ ски́нїи свидѣ́нїѧ.
И҆ сотвори́ша сы́нове і҆и҃лєвы по всѣ̑мъ, є҆ли̑ка заповѣ́да гдⷭ҇ь мѡѷсе́ю и҆ а҆арѡ́нꙋ, та́кѡ сотвори́ша.
